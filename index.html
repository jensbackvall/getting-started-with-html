The Web
The World Wide Web... what is it really?
As the name suggests, it is a kind of web, where millions of computers are linked up by web-like connections through wires, fibres, sattelites etc.
When I turn on my computer and want to access a web page, my computer recieves an HTML document that the browser I am using can interpret to show me whatever content that particular page has to show me.
You could say that the line of command looks something like this:

PERSON<->COMPUTER<->INTERNET<->HTTP<->SERVERS


In this example, HTTP is a type of protocol used to connect with servers, and servers are, as we all know, a kind of computers, often holding a whole or parts of a web page, online game etc.

Hyperlinks
Web pages contain hyperlinks and as these can point me toward anything, be it pictures, videos, text, music or any combination of these things, one can easily say that the anything can exist on the web, in one form or another.

HTML
HTML or Hyper Text Markup Language is the basic language of the internet. It contains:

HYPERLINKS, references to other content
TEXT content, which is basically what you read
MARKUP, defining what the above looks like
and it is a LANGUAGE



Tags
The Markup is made up of a load of tags, defining the way the content looks and what it does.
Most tags have an opening and a closing tag which show where it starts and where it ends. If the opening tag is <p>, the ending tag will be </p>. All tags start with a < and end with a >. Thank you to boutell.com for showing me how to write these in my code without the being a functional tag :-).
An opening and a closing tag, with text content in between make up what is referred to as an element.

Basic tags are the b tag, which makes text come out bold on the screen and the em tag, which makes text show up in italics.

Some tags are so called VOID tags. This means that they do not have an ending tag. An example is the IMG tag. The IMG tag uses the URL to an image and displays it on the page. Here's one that I found with a quick google search. Thanks to imbreannarose.com for this:



Whitespace
If you want whitespace on your page, you need to use specific tags for this, as HTML only gives you one space, no matter how many clicks of the spacebar you put into your code.
The one which may the most commonly used (but since my knowledge is limited, I am probably wrong about this) br tag, which gives you a line-break. The br tag has no ending tag, and using one simply gives you the afore-mentioned line-break.
You can also use the p tag to create a paragraph. This has an opening and an ending tag and everything in between them is a part of the paragraph.
the br tag is a so called inline tag and the p tag is a so called block tag. As I understand it, the inline tag works on the line of text it is in while the block tag creates a kind of invisible box for it's content.
Another block element is the div tag while another inline element is the span tag. I am currently not quite sure how to use these, so I'm sure more will come up later :-).
